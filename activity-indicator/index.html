Text Activity Indicator

Create your own custom scrolling text activity indicator with this little snippet. Set the text, size, speed, and color for a custom scrolling message. makeTestIndicator() returns both a DOM node and a setter method for setting the current state of the indicator. makeTextIndicatorWithBlockout() does the same, but also places a white screen over the page to prevent additional button clicks.
